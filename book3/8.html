Model Name: Weather Station with IoT Integration Working Principle: The model uses Arduino, sensors (temperature, humidity, and pressure), and an IoT module to collect weather data and send it to a cloud server. Materials Required: Arduino Uno, DHT11 (temperature and humidity sensor), BMP180 (pressure sensor), ESP8266 (WiFi module), and a breadboard. DIY Making Procedure: a. Connect the DHT11 and BMP180 sensors to the Arduino Uno. b. Attach the ESP8266 module to the Arduino for IoT communication. c. Program the Arduino to read data from the sensors and send it to a cloud server using IoT protocols (e.g., MQTT). d. Visualize the weather data on a cloud platform or a web-based dashboard. Why Is The Model Useful: This model introduces students to IoT and its application in weather monitoring for real-time data collection. Tips for Presentation: Discuss the importance of weather monitoring and how IoT-based weather stations can contribute to accurate weather predictions.
